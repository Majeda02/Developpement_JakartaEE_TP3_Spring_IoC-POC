### "# Developpement_JakartaEE_TP3_Spring_IoC-POC" 
#### Objectif :
Mettre en place plusieurs stratégies d’injection et de sélection d’implémentations IDao sans changer la classe métier MetierImpl ni la classe d’exécution, afin de respecter la POC/OCP.
Mécanismes abordés :
   * Profils Spring (dev, prod, file, api)
   * Priorité via @Primary
   * Alias via un @Bean de configuration
   * Sélection par propriété externe (@PropertySource + @Value)
